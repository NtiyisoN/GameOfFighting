#项目
###1.打开方式
找到proj.win32文件夹下，HelloCpp.sln。然后使用VS2010打开即可，同时配置Cocos2d-x2.x版本，3.x是编译不通过的!    
###2.代码路径
主目录Classes文件夹下，具体文件可看博文内容   
###3.资源路径
主目录Resources文件夹下
